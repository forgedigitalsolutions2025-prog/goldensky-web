# Images Required

To complete the website setup, you need to add the following images. You can download them from https://shadharawellness.com/

## Required Images

### Main Directory (`public/images/`)
- `logo.png` - Website logo
- `hero_bg_1_21.jpg` - Hero slider image 1
- `hero_bg_1_12.jpg` - Hero slider image 2
- `girl.jpg` - About page image

### Treatments Directory (`public/images/treatments/`)
- `service_1_1.jpg` - Body Massage
- `service_4_1.jpg` - Steam Bath
- `service_4_2.jpg` - Facial Treatment
- `service_1_3.jpg` - Head Massage / Foot Treatments
- `service_3_3.jpg` - Shirodhara
- `service_1_7.jpg` - Herbal Bath

### Testimonials Directory (`public/images/testimonials/`)
- `testi_1_2.jpg` - Testimonial image 1
- `testi_1_3.jpg` - Testimonial image 2

## How to Get Images

1. Visit https://shadharawellness.com/
2. Right-click on images and "Save Image As..."
3. Save to the appropriate directory listed above
4. Ensure filenames match exactly as listed

## Alternative

If you have access to the original website's files, you can copy images directly from their `images` folder.

## Note

The website will still function without images, but placeholders or broken image icons will appear. All image paths are configured to gracefully handle missing images.
